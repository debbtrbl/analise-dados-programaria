# 📊 Análise de Dados - DataHackers 2022

Este projeto foi desenvolvido durante o curso de Análise de Dados da PrograMaria, com o objetivo de explorar e extrair insights a partir da pesquisa anual DataHackers 2022.

## 📌 Sobre o dataset

A pesquisa DataHackers é uma das principais fontes de informação sobre o mercado de Tecnologia da Informação (TI) no Brasil, abordando temas como:
- Perfil dos profissionais
- Salários
- Experiência
- Educação
- Diversidade e inclusão

## 🔍 Etapas do projeto

### 📊 Análise Exploratória de Dados (EDA)
- Limpeza e tratamento dos dados
- Análise de distribuições
- Identificação de padrões e tendências
- Visualizações com gráficos

### 🤖 Modelagem Preditiva
Foi desenvolvido um modelo de **Regressão Linear** para analisar a relação entre salário e outras variáveis, como:

- Idade
- Gênero
- Tempo de experiência
- Nível de ensino
- Região
- Setor de atuação
- Número de funcionários da empresa
- Grau de satisfação

## 🛠️ Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Looker Studio

## 🎯 Objetivo

Praticar técnicas de análise de dados e machine learning, além de gerar insights relevantes sobre o mercado de TI no Brasil.

## 🚀 Resultados

O projeto permitiu identificar padrões importantes relacionados ao salário e perfil dos profissionais de TI, além de consolidar conhecimentos práticos em análise de dados.
